Whiteboard: Text Sizing Feature

The Text Sizing feature offers users the ability to: 1. Adjust Font Sizes: Increase or decrease font sizes as needed. 2. Preset Font Sizes: Choose from preset font sizes for convenience. 3. Custom Text Size: Input custom text sizes for enhanced clarity. This feature enhances readability, improves clarity, and establishes a structured visual hierarchy, ensuring well-organized and easily comprehensible content on the board.
