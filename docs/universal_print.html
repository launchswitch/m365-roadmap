Universal Print: Pull Print queues

Pull Print enables IT Pros to offer their end users a simplified view of the print solution. Specifically, IT Pros can set up virtualized printer(s) so that end users only need to know the name of one printer. Those users can securely release their own print jobs at any associated printer, depending on the setup.

Status: Launched | GA Date: June CY2025 | Preview Date: November CY2024 | Roadmap Status: Include this month
